【花蓮遊❷】林田山‧鳯林月盧梅子雞 @ 遇見"小眼睛"柯基~摩卡卡 :: 痞客邦 ::

2011-05-23

 

Day one 下午跟卡內家在後湖水月會和後，晚餐訂了月盧梅子雞

但時間還早，先去了『林田山林場』想外拍小忍犬們

一家三口擠上卡內家的車，摩卡卡自動喬好他的專屬位子

他這臉也太像 鳯蝶寶寶 了吧！笑噴～


 

林田山，在日治時代稱「摩里沙卡」。位於花蓮縣鳳林鄉的林田山林業文化園區

是東部最早開採的林區，從花蓮市區到林田山大約要1個小時

現在林田山的伐木事業已經停止，但這裡保留了大量的日式建築，所以來這拍忍犬很適合

可惜天氣不佳，一整個濕答答的，所以我們也懶得幫小鬼脫雨衣換忍犬服

不用出任務大家心情都不錯哦！   

 


結果林田山我也沒拍到什麼照片，哈哈，就介紹一下卡嚕比的Tiffany綠雨衣吧！ 

買了快一年完全沒穿過，因為基本上下雨根本就懶的出門

 


而且柯基身材很起乖，褲裝很難買到適合款，這是卡卡第一次穿褲子

變"開襠褲"爆出一撮屁毛超Q，小矮子褲管還得捲兩捲才不會拖地  

剛穿上他一度覺得卡卡的不太肯走

但我兒子適應力超強，三分鐘完全忘了這回事，跑跳全來！  

 


雨衣使用率不高，但很實用，上車前脫掉雨衣後又是枚清爽小狗

===========================

離開林田山，驅車前往鳯林月盧吃晚餐

梅子雞卡內麻已提前預訂，本來是訂土雞，臨時改為仿雞

 


因為天色昏暗，小DC拍出的效果實在很糟，戶外景觀就跳過

進入店內有著濃濃的古早味兒，櫃檯是用藥櫃當成

 

 


室內用餐區是日式的榻榻米，所以進入室內需要脫鞋

桌子跟裝潢相當復古，旁邊擺了兩個大茶壺，需要茶水請自取

 

在這麼徧遠的山區，用餐時刻座無虛席，大家都慕名而來吧！

連猛哥住當地的朋友都有特別推薦它

 

 

 事先已確認過，寵物是不可以進入餐廳的，所以我們自備了推車

小狗們可以安定休息，也不怕弄髒或被蟲蟲攻擊！

 


謝謝店家跟卡內麻事前的準備，給了個好位子，可以就近靠窗看顧小傢伙們

但萬萬沒想到最可怕的是當天用餐時段有一大群小孩

看到三隻狗坐推車跟看到派大星吃海綿寶寶一樣興奮，全部衝過來東戳西戳的，煩死了！ 

林母卡后走出去告誡後，一轟而散，不虧是老師，超有威嚴的！哈哈哈  

大家猜猜看卡卡一上推車在幹嘛？答對啦！就是睡覺～

我真的上輩子燒好香惹，這小子超級好帶啊！

因為醬董一直想衝出來跟小孩們一較高下？

所以幫他們倆個換車，以免又不小心在某處留情了 XDD

 

人多但上菜速度很有效率，點完菜沒多久就陸續上菜啦！    

 招牌必點－碳烤梅子桶雞，裡頭塞了香料跟鹹梅下去烤，但其實我沒有聞道梅香說 哈

不過雞肉紮實好吃，要沾著汁吃才不會太乾，再把桶底的香濃雞汁淋在飯上，超讚！

 

請醬拔幫忙手趴雞，忍著燙不敢叫出聲，so man！  

 

 6人份雙色米飯，白飯&紫米各半，看起來有養生到

 

 梅拌野蕨，用梅子主要調味，微酸微辣，非常開胃，這道我覺得必點！

 

 照燒山豬肉，又是道下飯的料理，口味很重，肉片很薄不膩口

 

 三杯杏鮑菇，連不愛菇類的猛哥都愛吃

 

鐵板牛柳，不知道是不是鐵板過熱，牛肉頗老，有點小失敗

 

五更腸旺，沒有太大特色，其實我已經忘了它的味道了 

　(我沒說是雷哥點的，我沒有！XD ) 

寫文時我有問他覺得好吃嗎？他說好像還好，他只吃了一口，哈哈哈！惡劣啊這人

所以來吃的捧油這道可以跳過，ㄎㄎㄎ

  

香酥一口蟹，鹹香鹹香一隻接一隻

 

山藥排骨湯，湯頭好讚，山藥也沒臭臭的土味，固的！

 


我們這次人夠多適合來吃，菜色可多點幾道，人太少來這不好點菜

晚上室外也有夜景可以看，可是天色暗，下山的路沒燈，沒有逗留太久就打道回民宿

為了吃這頓我們來回花了近兩小時，為了美食值得啦！

下次去花蓮玩的捧油，記得可以先訂位嚐嚐唄～
